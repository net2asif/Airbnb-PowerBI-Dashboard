# 🏡 Global Airbnb Performance Dashboard

## 📊 Overview
Power BI dashboard analyzing 279K listings 
across 10 global cities with 5.37M reviews.

## 🔑 Key Insights
- Entire Place drives 70%+ of all reviews
- Peak bookings in 2019 before COVID crash
- Mexico City leads all rating categories

## 🛠️ Tools Used
- Power BI Desktop
- DAX
- Data Modeling
